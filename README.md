# Beauty Care SLZ

Site institucional desenvolvido para divulgação de serviços de beleza e cuidados estéticos.

## Sobre o Projeto

O **Beauty Care SLZ** foi criado com o objetivo de apresentar de forma moderna e organizada os serviços oferecidos pelo salão, facilitando o contato com clientes e fortalecendo a presença digital da marca.

O projeto foi desenvolvido utilizando tecnologias web básicas, com foco em design limpo, navegação simples e boa experiência para o usuário.

## Acesse o Site

🌐 https://beautycareslz.com.br

## Funcionalidades

- Página inicial moderna e responsiva  
- Apresentação dos serviços  
- Informações sobre o salão  
- Área de contato  
- Navegação simples e intuitiva  
- Layout adaptado para celular e computador
