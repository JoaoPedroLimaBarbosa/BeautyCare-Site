💇‍♀️ Beauty Care — Salão de Beleza em São Luís (MA)

O Beauty Care é um site desenvolvido para apresentar os serviços de um salão de beleza localizado em São Luís do Maranhão. O projeto tem como objetivo oferecer uma presença digital moderna, elegante e responsiva, facilitando o acesso dos clientes às informações do salão.

✨ O site conta com:

* Apresentação dos serviços (cabelo, maquiagem, depilação, unhas e estética)
* Design moderno e intuitivo
* Layout totalmente responsivo (adaptado para celular, tablet e desktop)
* Navegação simples e agradável para o usuário

🚀 Tecnologias utilizadas:

* HTML5
* CSS3

📍 Localização:
São Luís - Maranhão, Brasil

💡 Objetivo do projeto:
Criar um site institucional simples, bonito e funcional para ajudar o salão a atrair mais clientes e fortalecer sua presença online.
